Установка:

1) Скачать проект с репозитория. Проект уже содержит в себе HydraAPI и HydraCore
2) Собрать HydraAPI и HydraCore которые находятся в dependencies/Hydra
3) Установить следующие пакеты:
sudo apt-get install libfreeimage-dev
sudo apt-get install mesa-common-dev libglu1-mesa-dev libglfw3-dev libglfw3
sudo apt install ocl-icd-opencl-dev
sudo apt-get install libsdl2-dev libsdl2-ttf-dev libsdl2-mixer-dev libsdl2-image-dev libglew-dev libboost-system-dev libboost-filesystem-dev libglm-dev
sudo apt-get install python-dev 

Из папки grade
4) Cmake CMakeLists.txt
5) make 

Запуск:

Демоверсия: ./main -demo <N> -hydra = <scene_name>

<N> - приблизительное количество деревьев (желательно не больше 100)
<scene_name> - название сцены. Сцена вместе с демо-изображением будет сохранена в hydra_scenes/<scene_name>

Прочие аргументы:
-no_debug - отключить дебажную печать 
-render ВМЕСТО -hydra = ... - демонстрация сцены во встроенном real-time рендере вместо сохранения Гидрой
